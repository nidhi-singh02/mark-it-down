Privacy Policy

Web to Markdown — Chrome Extension

What this extension does

Web to Markdown reads the HTML of the current web page solely to convert it to Markdown format. All processing occurs locally within your browser.

Data collection

This extension does not collect, store, or transmit any personal data or browsing information. There are no analytics, tracking, cookies, or third-party services. No data leaves your device.

Permissions

The extension requests only two permissions:

activeTab — to read the content of the current page when you click the extension icon.

scripting — to extract the page HTML for conversion to Markdown.

Chrome Web Store compliance

This extension's use of information received from Chrome APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
